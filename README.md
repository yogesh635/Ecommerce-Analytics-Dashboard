# Ecommerce-Analytics-Dashboard
# 📊 E-Commerce Sales Analytics Dashboard (Power BI)

## 🔍 Project Overview

This project presents an interactive **E-commerce Sales Analytics Dashboard** built using Power BI.
It provides insights into sales performance, profit trends, customer behavior, and regional distribution.

The dashboard allows users to explore data dynamically using filters and visualizations.

---

## 🚀 Features

* 📌 KPI Cards (Total Sales, Total Profit, Total Orders)
* 📈 Monthly Sales & Profit Trends
* 🛍 Sales by Category & Sub-Category
* 🌍 Region-wise Sales Distribution
* 💳 Payment Mode Analysis
* 🗺 City-wise Sales Visualization (Map)
* 🎛 Interactive Filters (Date, Category, Region, Payment Mode

---

## 📊 Dataset Information

* Format: Excel (.xlsx)
* Contains: Sales, Profit, Category, Region, Payment Mode, Date
* Cleaned and structured for analysis

---

## 🛠 Tools & Technologies Used

* Power BI (Data Visualization)
* Microsoft Excel (Dataset)
* Data Cleaning & Data Analysis

---

## 🧪 How HR Can Test This Project

You can interact with this project by following these steps:

1. Download the dataset file:
   `dataset/ecommerce_data.xlsx`

2. Modify the data:

   * Change sales values
   * Add/remove categories
   * Update regions or dates

3. Open the dashboard file:
   `dashboard/ecommerce_dashboard.pbix`

4. Click **Refresh** in Power BI Desktop

5. Observe how charts and KPIs update dynamically

---

## ⚠️ Requirements

* Power BI Desktop must be installed to open `.pbix` file

---
